
# Projecto PadelPro Manager (pager) 🎾

PadelPro Manager (pager) es un sistema desarrollado para optimizar la administración de centros deportivos que operan bajo alta demanda, el sistema reemplaza los registros manuales por una plataforma integral alojada en la nube



## 📋 Requisitos para Instalación

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```
    
## 🚀 Características

- Gestión de Reservas: Control de horarios en bloques fijos de 90 minutos.

- Disponibilidad en Tiempo Real: Consulta visual de canchas libres.

- Control de Pagos: Registro y validación de referencias de pago móvil y transferencias.

- Roles de Usuario: Acceso diferenciado para Jugadores, Recepcionistas y Administradores.

- Reportes: Análisis de ocupación y rendimiento financiero.
## 🛠️ Tecnologías Utilizadas

- Lenguaje: PHP 8.x.

- Base de Datos: MySQL.

- Frontend: HTML5, CSS3 (Tailwind CSS), JavaScript.

- Paradigma: Programación Orientada a Objetos (POO).
## 🔧 Instalación y Configuración

1) Clonar el repositorio:

Bash
git clone https://github.com/Ginosaurio04/pager-2.git 

2) Configurar la Base de Datos:

- Abre phpMyAdmin.

- Crea una base de datos llamada pager.

- Importa el archivo database.sql (o el nombre que tenga tu script SQL) ubicado en la carpeta del proyecto.

3) Configurar Conexión:

- Edita el archivo de conexión (ej. db.php o conexion.php) con tus credenciales locales:

PHP
$host = "localhost";
$user = "root";
$pass = "";
$db   = "pager";

4) Ejecución:

- Copia la carpeta del proyecto en htdocs (si usas XAMPP).

- Accede desde tu navegador a http://localhost/pager.
